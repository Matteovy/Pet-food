EN 
There are four data sets: 
1. retailer.csv: In this table, you'll find the brand's transactional data. 
2. tv_publisher.csv: In this table, you'll find the brand's television ads data.
3. programmatic_publisher.csv: In this table, you'll find data from the brand's programmatic advertising (i.e. on the web).
4. socio_demo.csv: In this table you'll find socio-demo information on people exposed to programmatic media.
5. mapping_transac_publisher_tv.csv is the mapping table used to connect the various tables. 

You will find the table dictionary in the TO_READ_dict.csv file.
